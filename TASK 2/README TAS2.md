# Task 2 - Static Website Hosting using Amazon S3

## Objective
Host a static website using Amazon S3.

## Tools Used
- AWS S3
- HTML

## Steps Performed
1. Created an S3 bucket.
2. Uploaded index.html.
3. Enabled Static Website Hosting.
4. Disabled Block Public Access.
5. Added Bucket Policy.
6. Accessed the website using the S3 website endpoint.

## Website Endpoint

http://akhil-task2-static-website-2026.s3-website.eu-north-1.amazonaws.com

## Result

Successfully hosted a static website using Amazon S3.
